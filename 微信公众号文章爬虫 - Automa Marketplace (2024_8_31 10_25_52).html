Make sure you check the content of the workflow before executing it. And If you found something suspicious, report it.

If the workflow trigger is interval, on a specific date, keyboard shortcut, etc. To make it work, you need to open the workflow first and then click the save button.

需要有一个公众号账号，以便访问公众号后台、查询其他公众号的文章数据。

为避免触发限制，设定了相对慢一些的爬取速度，约 5 篇文章/分钟，但运行期间完全可以去做其他事。

使用步骤：

  1. 安装这个工作流（点击页面上的 Add to extension，不是 Downvote）

  2. 在网页端登录微信公众号，保留登录后的主页打开

  3. 运行这个工作流，输入要爬取的公众号名称、向前爬取的截止日期（可选）

  4. 运行至搜索公众号，在结果中手动选择你要爬取的公众号

  5. 等待数据收集结束

因公众号后台接口存在访问限制，如爬取中遇到数据不刷新，推荐等待一段时间后，结合这些做法提高成功率：

  • 设一个更大的翻页间隔，默认 1 分钟翻一次（编辑模式下 60000ms 的那个块）

  • 指定一个更近的爬取截止日期
